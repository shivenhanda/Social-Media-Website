# Social-Media-Website

## 📝 Description

Social-Media-Website is a comprehensive web-based social networking platform powered by MERN stack (MongoDB, Express.js, React.js, Node.js). Designed for seamless user interaction, the application features a robust authentication system to ensure data privacy and secure access. It offers a streamlined web interface that enables users to connect, share content, and manage their social presence effectively within a modern digital environment.

## ✨ Features

- 🔐 JWT Authentication & Authorization
- 👤 User Registration & Login
- 📝 Create and Manage Posts
- 🌐 REST API Integration
- ⚡ Responsive React Frontend
- 🔒 Password Hashing with bcrypt
- 📦 MongoDB Database Integration

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT
- bcrypt

# Clone the repository
git clone https://github.com/shivenhanda/Social-Media-Website.git

# Install dependencies
npm install

## ▶️ Run Commands

### 🔧 Start Backend

```bash
cd backend
npm install
node script.js

```
### 🔧 Start Frontend

```bash
cd frontend
npm install
npm run dev

```
